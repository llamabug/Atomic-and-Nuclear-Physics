# Atomic-and-Nuclear-Physics
A website where you can practice questions for AS91172- a nz year 12 exam on nuclear/atomic physics
<br><br>
I needed practice, but I wanted to code, so why not combine?
<br><br>
Features 3 different subtopics, quiz questions in both multiple choice and input styles, and a page of additional resources.
<br><br>
Originally, this was going to be multi-choice quizzes with notes at the bottom, but I (a) wanted to make it harder, thus making the input questions and (b) ended up not wanting to rewrite all my notes online, so that plan was scrapped and replaced with a resources page.
<br><br>
Happy studying!
